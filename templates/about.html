Elderly People Monitoring System

About

Elderly People Monitoring System (EMS) is a project designed to monitor and ensure the well-being of elderly individuals. The system utilizes computer vision and deep learning techniques to detect falls and analyze emotional states through facial expressions.

Project Details

  • Technology Stack: Python, Flask, OpenCV, PyTorch, DeepFace, TensorFlow
  • Features:
    • Real-time fall detection using computer vision
    • Emotion analysis through facial expressions
